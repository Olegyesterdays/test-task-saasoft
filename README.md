# Тестовое задание для SaaSoft

Необходимо реализовать форму управления учетными записями.

## Стек технологий

- Vue.js 3 (Composition API)
- TypeScript
- Pinia
- Vuetify
- Docker compose

## Запуск приложения

### Production режим

Запуск приложения

``` bash
npm start
```

### Dev режим

Установка зависимостей

``` bash
npm i
```

Запуск приложения

``` bash
npm run dev
```
